API Documentation

GET /movies

Returns a list of all movies.

Method: GET

URL: /movies

Response Format:

[
    {
        "Title": "The Shawshank Redemption",
        "Description": "Two imprisoned men bond...",
        "Genre": {
            "Name": "Drama",
            "Description": "A genre that focuses on..."
        },
        "Director": {
            "Name": "Frank Darabont",
            "Bio": "Frank Darabont is an..."
        },
        "ImageURL": "https://...",
        "Featured": true
    },
    ...
]

GET /movies/:title

Returns details of a specific movie by title.

Method: GET

URL: /movies/:title

Parameters:

title (string): The title of the movie.

Response Format:

{
    "Title": "The Shawshank Redemption",
    "Description": "Two imprisoned men bond...",
    "Genre": {
        "Name": "Drama",
        "Description": "A genre that focuses on..."
    },
    "Director": {
        "Name": "Frank Darabont",
        "Bio": "Frank Darabont is an..."
    },
    "ImageURL": "https://...",
    "Featured": true
}

GET /movies/genre/:genreName

Returns details of movies by genre name.

Method: GET

URL: /movies/genre/:genreName

Parameters:

genreName (string): The name of the genre.

Response Format:

{
    "Name": "Drama",
    "Description": "A genre that focuses on..."
}

POST /users

Creates a new user.

Method: POST

URL: /users

Request Body:

{
    "name": "Kim"
}

Response Format:

{
    "id": "1",
    "name": "Kim",
    "favoriteMovies": []
}

PUT /users/:id

Updates a user's name by their ID.

Method: PUT

URL: /users/:id

Parameters:

id (string): The ID of the user.

Request Body:

{
    "name": "Joe"
}

Response Format:

{
    "id": "1",
    "name": "Joe",
    "favoriteMovies": ["The Godfather"]
}

PUT /users/:id/:movieTitle

Adds a movie to a user's favorite list.

Method: PUT

URL: /users/:id/:movieTitle

Parameters:

id (string): The ID of the user.
movieTitle (string): The title of the movie to be added.

Response Format:

"The Godfather has been added to user 1's favorite movies list."

DELETE /users/:id/:movieTitle

Removes a movie from a user's favorite list.

Method: DELETE

URL: /users/:id/:movieTitle

Parameters:

id (string): The ID of the user.
movieTitle (string): The title of the movie to be removed.

Response Format:

"The Godfather has been removed from user 1's favorite movies list."

DELETE /users/:id

Deletes a user by their ID.

Method: DELETE

URL: /users/:id

Parameters:

id (string): The ID of the user.

Response Format:

"user 1 has been deleted."
