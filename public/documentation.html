API Documentation

Returns a list of all movies.

Method: GET

URL: /movies

Response Format:

[
    {
        "_id": "64f62e5f2b5e6d62e3b2f142",
        "Title": "The Shawshank Redemption",
        "Description": "Two imprisoned men bond...",
        "Genre": {
            "Name": "Drama",
            "Description": "A genre that focuses on..."
        },
        "Director": {
            "Name": "Frank Darabont",
            "Bio": "Frank Darabont is an..."
            "Birth": "1959-01-28"
        },
        "ImageURL": "https://...",
        "Featured": true
    },
    ...
]

Method: GET

Response Format:

POST /users

Creates a new user.

Method: POST

URL: /users

Request Body:

{
    "Username": "Kim",
    "Password": "password123",
    "Email": "kim@example.com",
    "Birthday": "1990-01-01"
}

Response Format:

{
    "_id": "64f62e5f2b5e6d62e3b2f142",
    "Username": "Kim",
    "Email": "kim@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
}

PUT /users/:Username

Updates a user's info by their Username.

Method: PUT

URL: /users/:Username

Parameters:

Username (string): The Username of the user.

Request Body:

{
    "Username": "Kim",
    "Password": "newpassword",
    "Email": "kim@example.com",
    "Birthday": "1990-01-01"
}

Response Format:

{
    "_id": "64f62e5f2b5e6d62e3b2f142",
    "Username": "Kim",
    "Email": "kim@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
}

PUT /users/:Username/movies/:MovieID

Adds a movie to a user's favorite list.

Method: PUT

URL: /users/:Username/movies/:MovieID

Parameters:

Username (string): The Username of the user.
MovieID (string): The ID of the movie to be added.

Response Format:

{
    "_id": "64f62e5f2b5e6d62e3b2f142",
    "Username": "Kim",
    "Email": "kim@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": ["64f62e5f2b5e6d62e3b2f143"]
}

DELETE /users/:Username/movies/:MovieID

Removes a movie from a user's favorite list.

Method: DELETE

URL: /users/:Username/movies/:MovieID

Parameters:

Username (string): The Username of the user.
MovieID (string): The ID of the movie to be removed.

Response Format:

{
    "_id": "64f62e5f2b5e6d62e3b2f142",
    "Username": "Kim",
    "Email": "kim@example.com",
    "Birthday": "1990-01-01",
    "FavoriteMovies": []
}

DELETE /users/:Username

Deletes a user by their Username.

Method: DELETE

URL: /users/:Username

Parameters:

Username (string): The Username of the user.

Response Format:

"User 'Kim' has been deleted."
